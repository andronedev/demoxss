Démonstration des attaques XSS

Ces exemples présentent les trois types d'attaques XSS : réfléchi, stocké et DOM-based. Veuillez suivre les instructions pour chaque démonstration.

  • XSS réfléchi

    1. Ouvrez le lien suivant dans un nouvel onglet : reflected.html

    2. Ajoutez le texte suivant à la fin de l'URL dans la barre d'adresse : ?message=<script>alert('Attaque XSS réfléchie!')</script>

    3. Appuyez sur Entrée pour déclencher l'attaque XSS réfléchie.

  • XSS stocké

    1. Ouvrez le lien suivant dans un nouvel onglet : stored.html

    2. Entrez le texte suivant dans le champ de commentaire : <script>alert('Attaque XSS stockée!')</script>

    3. Cliquez sur "Submit" pour déclencher l'attaque XSS stockée.

  • XSS DOM-based

    1. Ouvrez le lien suivant dans un nouvel onglet : dom_based.html

    2. Entrez le texte suivant dans le champ de texte : <script>alert('Attaque XSS DOM-based!')</script>

    3. L'attaque XSS DOM-based se déclenche automatiquement lors de la saisie du texte.
