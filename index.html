Démonstration des attaques XSS

Instructions

  1. Pour la démonstration de l'attaque XSS réfléchie, cliquez sur le lien suivant et ajoutez un paramètre ?search=<script>alert('XSS')</script> à la fin de l'URL :

    Démonstration de l'attaque XSS réfléchie

  2. Pour la démonstration de l'attaque XSS stockée, utilisez le formulaire pour soumettre un commentaire contenant un script, par exemple : <script>alert('XSS')</script>.

    Démonstration de l'attaque XSS stockée

  3. Pour la démonstration de l'attaque XSS basée sur le DOM avec une calculatrice, saisissez une expression mathématique ou un script malveillant, par exemple : 1+1;alert('XSS');.

    Démonstration de l'attaque XSS basée sur le DOM avec calculatrice
